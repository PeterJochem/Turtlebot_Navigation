Detailed Description

Implements methods for storing and updating a robot's odometry data from published encoder data.

Publishes: /odom, /visualization_marker

Subscribes: /joint_states

Services: /set_pose - Reset the robot to a new SE(2) configuration High level abstraction and trakcing of a robot's odometry
