Transform2D rigid2d::Transform2D::integrateTwist ( Twist2D  )

Integrate a constant twist for unit time So, if I am T_sb and I apply the given twist for 1s then I end up at T_bc. So, if I wanted T_sc, then I would need to multiply T_sb * T_sc.

Twist2D rigid2d::Transform2D::operator() ( Twist2D  twist)

Uses adjoint to map a twist from one frame to another See page 85 of Modern Robotics for more details. Since this is planar motion, the adjoint simplifies down to a pretty nice little set of expressions. Initially, I constructed the whole 6x6 matrix but decided this was likely a better implementation.

Parameters
twist_original- the input twist we are doing a change of frame on
Returns
the twist in the new frame
